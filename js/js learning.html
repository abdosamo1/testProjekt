JavaScript Learning

In JavaScript, you can declare variables using the 'var', 'let', or 'const' keywords. Variables in JavaScript are dynamically typed, meaning you don't need to declare their type before using them. Here's an example of declaring a variable using 'var':
var myVariable = "Hello, World!";

You can also declare a variable using 'let' or 'const' in ES6:
let myVariable = "Hello, World!";

Variables declared using 'const' are immutable, meaning you can't change their value after they're initialized:
const myVariable = "Hello, World!";

You can access and modify the value of a variable by using its name
